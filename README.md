## 🔗 Usage

Grab any GIF's raw link directly from this repo, or right-click → "copy image address" from the gallery above, then use it anywhere:

```md
![gif](DIRECT-LINK-HERE)
```

```html
<img src="DIRECT-LINK-HERE" width="380"/>
```

<br/>

## ➕ Contributing

Got a good GIF to add? Contributions are welcome.

1. Fork this repo
2. Add your GIF link inside the gallery table in `README.md` (or upload the file itself to a `/gifs` folder)
3. Open a Pull Request with a short description of the GIF
