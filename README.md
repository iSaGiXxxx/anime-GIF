## How to Add a GIF [🔝](#menu)

1. Find or grab your GIF's direct link (e.g. from Giphy: right-click the gif → "Copy image address")
2. Paste it under the right category above using:
   ```html
   <img src="DIRECT-LINK-HERE" width="260"/>
   ```
3. Keep 2–4 gifs per row using `&nbsp;` between `<img>` tags for spacing
4. Commit your changes
